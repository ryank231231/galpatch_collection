## About the project
The project aims to collect galgame localization patches.

該計劃旨在收錄 Galgame 的（非官方）本地化補丁（e.g：漢化補丁）

## What patch do we collect
- 所有補丁按照**原樣**收錄，意味者不能對補丁壓縮包內文件作出修改。
- 我們收錄**所有**漢化組發佈的補丁。唯一的要求是：補丁能在所有人的設備上不收費地運行。
